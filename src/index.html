Setup

  1. create a folder for your project

  2. create README.md file

  3. run on Terminal

    nmp init

    (after the command above the package.json file will be created)

  4. Edit package.json file to add your project's name 
    {
    "name": "[project name]",
    ...
    "author": "[your name]"
}

  5. create .gitignore file and skipt this folders

    dist/

    src/styles

    node_modules
